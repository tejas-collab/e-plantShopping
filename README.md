# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React and Redux-based web application developed for Paradise Nursery. The application allows users to browse a variety of houseplants, add them to a shopping cart, manage quantities, and view the total cost before checkout.

## Features

- Display houseplants grouped into multiple categories
- Each plant includes:
  - Thumbnail image
  - Plant name
  - Price
- Add to Cart functionality
- Disable "Add to Cart" button after adding item
- Dynamic cart icon with item count
- Increase and decrease product quantity
- Remove items from cart
- Display total cart value
- Checkout button (Coming Soon)

## Purpose

The purpose of this project is to demonstrate the use of:
- React components
- Redux state management
- Dynamic UI updates
- Shopping cart logic implementation

## Usage

1. Open the landing page.
2. Click on "Get Started".
3. Browse plants by category.
4. Add plants to the cart.
5. Manage quantities in the cart page.
6. View total cost and proceed to checkout.
